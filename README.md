# Pong

Juego clásico de Pong desarrollado con WebComponents nativos y Canvas 2D, sin frameworks ni bundlers.

## Requisitos

- [pnpm](https://pnpm.io/)

## Desarrollo

```bash
pnpm dev
```

## Cómo jugar

- **Jugador 1:** `W` / `S`
- **Jugador 2:** `↑` / `↓`
- **Espacio:** pausar / reanudar / reiniciar

## Estructura

- `src/components/` — WebComponents
- `src/modules/` — lógica del juego (board, ball, bar, collision, renderer)
- `src/css/` — estilos globales
